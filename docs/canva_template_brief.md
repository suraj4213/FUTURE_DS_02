## Canva Report Template Brief (Optional)

### Purpose
Create a polished stakeholder-ready summary that reuses visuals exported from Power BI/Looker Studio and highlights the strategic takeaways.

### Brand Direction
- Palette: Gradient from #5E4AE3 → #7B6CF6 → #A6C1FF, accent #1F2A44, neutral #F5F7FF.
- Typography: Headings Montserrat Bold, body text Roboto Regular.
- Icon set: Simple line icons for metrics (Impressions, Clicks, Spend, Revenue).
- Include company logo placeholder on each page.

### Page Breakdown
1. **Cover**  
   Title “Social Media Campaign Performance Report” + period (e.g., “Q1 2025”) + background gradient.
2. **Executive Summary**  
   - Four KPI cards (Impressions, Spend, Revenue, ROI).  
   - Short paragraph + 3 bullet highlights.  
   - Space for hero visual (Power BI line chart screenshot).
3. **Platform Performance**  
   - Three columns (Facebook, Instagram, Twitter) with mini-cards for ROAS, ROI, Engagement.  
   - Mini bar or donut charts.
4. **Top Campaigns**  
   - Card layout with campaign name, objective, spend, ROAS, key insight.  
   - Optional success story callout.
5. **Insights & Recommendations**  
   - Two-column layout (Findings vs Actions).  
   - Add icons for urgency/impact.  
   - Include notes on budget reallocation, target audiences, and creative guidance.

### Assets to Insert
- PNG exports of Power BI visuals (use transparent background if possible).
- Tables/heat maps from Looker Studio or Excel for deeper detail.
- Icons (e.g., Canva Elements) representing objective types, demographics, funnel stages.

### Export
- Download PDF (high quality) for distribution.
- Export PNG slides for social posts or presentation decks.
- Use Presentation mode for live walk-throughs.

### Handoff Checklist
- Update text placeholders with current period.
- Replace dummy logos with actual brand assets.
- Store template at `dashboards/Canva/` (link recorded in `reports/insights.md`).
